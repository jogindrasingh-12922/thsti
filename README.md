# THSTI .NET Version

This is the complete, cleaned project migrated entirely to ASP.NET Core 8 Web API.

## Backend (server)
- ASP.NET Core 8 Web API.
- Uses EntityFramework Core with PostgreSQL.
- Run `dotnet run` inside `server`.

## Frontend (thsti-web & admin)
- React / Vite apps.
- Copied with updated `.env` pointing to .NET API.
- Run `npm install` then `npm run dev` inside each.
